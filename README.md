# <img src="https://github.com/user-attachments/assets/49dfd1b8-cc8d-4220-890b-0cc589ca30ad" width="40"/> e-SparX 

e-SparX (the **sp**ace for **ar**tifact e**x**change in **e**nergy ML research) is a software application that facilitates sharing and finding ML artifacts for fast and easy reuse, specifically designed for the needs of energy ML researchers. It features a web-based GUI that organizes and visualizes ML artifacts in interconnected ML pipeline graphs, enabling the finding and benchmarking of ML projects from the energy community. To publish ML artifacts with minimal effort, e-SparX offers a user-friendly package to publish ML artifacts with minimal effort.

🪄 Go ahead and check out the first available pipelines on the [e-SparX Website](https://e-sparx.emt.ed.tum.de/)!

> **Note**: You must be connected to the TUM network to access the website. We will host e-SparX publicly soon.
